本文件中B树的版本为国内教科书作者严蔚敏提供的版本，和文件夹 balance_tree 中定义的B树版本有所不同。

该版本的特点为：（B树的阶为M）
1. 树的每个节点最多有M棵子树, 也就是每个节点中的元素最多有M-1个。
2. 根节点如果不是叶子节点的话，那么最少可以有2棵子树，也就是只有一个元素。
3. 除了根之外，所有节点(包含叶子节点)最少有M/2（向上取整）棵子树, 也就是除了根节点，所有其他节点
   的元素数最少为M/2（向上取整）-1 个节点。
4. 所有的叶子节点的深度都相同。
5. 所有节点均有序排列。

通过上面B树特征的分析，M最小是3，如果存在一个2阶B树，里面只有2个元素的话，那么将不满足特点2中根有
2棵子树的要求，与定义相矛盾。
（ balance_tree 文件夹中定义的B树版本最小阶可以是2）
