本文件夹 binary_avl_tree 内，实现了AVL树6个版本的代码，分别是:

1. 含有高度hight, 没有parent_指针的递归插入，删除版本
2. 含有高度hight, 没有parent_指针的非递归插入，删除版本

4. 含有平衡因子，没有parent_指针的非递归插入，删除版本

5. 含有平衡因子，携带parent_指针的递归插入，删除版本
6. 含有平衡因子，携带parent_指针的非递归插入，删除版本

其中5, 6版本的代码文件中额外实现了c++的四组拷贝构造函数以及赋值运算符重载函数（含右值引用版本），因为携带
parent_指针，可以做到无堆栈协助地遍历整棵树，所以作为tools 目录中map和set容器的一种实现（采用6号非递归版本）。
